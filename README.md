# sleep-delay

[![npm version](https://img.shields.io/npm/v/sleep-delay.svg)](https://www.npmjs.com/package/sleep-delay)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Async sleep and delay utilities for timing control

## Installation

```bash
npm install sleep-delay
```

## Functions

- `sleep` — Async sleep for specified milliseconds
- `delay` — Delay a function execution by ms
- `timeout` — Wrap a promise with a timeout
